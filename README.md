# Бот гороскоп

Это простой Telegram-бот, который показывает гороскоп по знакам зодиака.

---

## Что умеет

- Показывает список знаков зодиака
- Выдаёт прогноз по выбранному знаку
- Есть кнопка обновления данных
- Данные хранятся в SQLite

---

## Как устроено

- bot.py — запуск бота
- handlers — обработка кнопок
- keyboards — кнопки
- db.py — работа с базой
- horo.py — парсер данных
- services — логика получения прогноза

---

## Как запустить

1. Установить зависимости: pip install -r requirements.txt
2. Создать файл `.env`: BOT_TOKEN=твой_токен
3. Запустить бота: python bot.py

---

## Обновление данных

Если нужно обновить гороскоп:
python update_db.py


---

## База данных

Используется SQLite файл `horoscope.db`.

Таблица:
- name — знак зодиака
- forecast — прогноз
